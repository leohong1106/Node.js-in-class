Node.js Package Manager

    about npm

  • 노드를 위한 패키지 매니저
  • 다른 개발자들이 만든 확장 모듈을 설치하고 관리해 주는 기능
  • npm 사이트에 올라와 있는 노드 모듈을 패키지, 확장 모듈이라고 함
  • 하나의 모듈을 사용하기 위해서는 그 모듈이 사용하는 다른 모듈들도 같이 설치해 줘야 함
  • 이러한 의존성 관리도 npm이 수행해 줌
  • 확장 모듈 개발자는 자신의 모듈을 npm 중앙 저장소로 배포하고
  • 확장 모듈 사용자는 해당 모듈을 설치하고 package.json 파일에 기록해 둠
  • 다른 사용자와 협업을 할 때는 package.json 파일만 공유하면 됨

    yarn패키지 매니저

  • npm은 노드진영에서 배포한 패키지 매니저
  • 페이스북에서 npm을 개선해서 만든 패키지 매니저가 yarn패키지
